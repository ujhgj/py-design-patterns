# Python-Flavored Design Patterns

## venv

Best to use venv to run a dedicated environment:
1. Create environment:
    ```bash
    python3 -m venv venv
    ```
1. Activate:
    ```bash
    source venv/bin/activate
    ```
1. Install dependencies:
    ```bash
    pip install -r requirements.txt
    ```

## Tests

python -m pytest
